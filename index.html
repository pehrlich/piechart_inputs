PieChart Inputs

  • Click or drag to change values
  • Mobile compatible
  • Coffeescript Implementation
  • Zip Archive: piecharts_inputs-master.zip
  • Project Page: pehrlich/piecharts_inputs

Demo:

Markup:

        <form id="pie">
            <input type="hidden" name='author' value="20">
            <input type="hidden" name='publisher' value="2">
            <input type="hidden" name='charity' value="3">
        </form>

        <script type="text/javascript">
            $('#pie').pie();
        </script>

Options:

label_radius number of pixels inside from the outer edge of the pie to place the numbers. Negative values OK. Default: 40
label_prefix String prefix for all labels. Default: '$'
label_suffix String suffix for all labels. Default: ''
color color scheme for the pie slices. Should accept an integer and return anything accepted by the 'fill' attribute. Default: d3.scale.category20();
display_accuracy Number of decimals to show over the pie slices. Default: 2
output_accuracy Number of decimals to put in to the field inputs. Default: 2
